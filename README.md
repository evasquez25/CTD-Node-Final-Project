# CTD Node Final Project

A full-stack financial management application built with React, Node.js, and MongoDB.

## Features

- **User Authentication**: Registration, login, and JWT-based session management
- **Debt Management**: Create, read, update, and delete debts with automatic balance calculations
- **Payment Management**: Track both bills and debt payments with automatic debt balance updates
- **Modern UI**: Responsive design with React components and CSS modules

## Tech Stack

- **Frontend**: React, Vite, CSS Modules
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Authentication**: JWT tokens, bcrypt password hashing
- **Security**: Helmet, XSS protection, CORS configuration

## Deployment

This project is configured for deployment on Render.com with both frontend and backend services.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Production

```bash
# Start production server
npm start
```
